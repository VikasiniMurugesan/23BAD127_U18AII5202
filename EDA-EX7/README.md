**Objective**

To reduce visual clutter in large datasets using effective visualization techniques in R.

**Dataset**

File: 7.social_media_interactions.csv
Contains social media engagement metrics such as Likes, Shares, Comments, and Engagement_Score.

**Tools Used**
R (v4.5.2)

ggplot2

dplyr

**Techniques Applied**
Raw Scatter Plot (to observe over-plotting)

Alpha Blending

Jittering

Aggregation

2D Binning

**Conclusion**

Over-plotting hides patterns in dense datasets. Techniques like alpha blending and binning improve clarity and support better analytical insights.
